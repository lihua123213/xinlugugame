# 像素农场

基于 HTML5 Canvas 的像素风农场经营游戏，全部视觉元素由 Canvas API 逐像素绘制，无外部图片依赖。

## 游戏内容

- 80x35 格大地图，含河流、森林、山脉、建筑
- 6 种作物种植（防风草/土豆/番茄/玉米/小麦/南瓜）
- 4 个季节轮换，日夜光照系统
- 僵尸地牢战斗，击败 Boss 获取粉色宝石
- 3 个 NPC 交互 + 好感度送礼系统
- 饭店、铁匠铺、矿洞、钓鱼小游戏
- 局域网双人联机（P1 权威模式）
- localStorage 存档/读档

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 游戏画面渲染，960x640 分辨率 |
| 原生 JavaScript (ES6) | 游戏逻辑，class 语法模块化 |
| Node.js + ws | WebSocket 联机服务器 |

## 运行方式

### 单人模式

直接双击 `farm-game.html` 即可在浏览器中运行。

### 联机模式

```bash
npm install
node server.js
```

- P1（主机）：`http://localhost:8080`
- P2（客户端）：`http://<局域网IP>:8080/client.html`

### 操作说明

| 按键 | 功能 |
|------|------|
| WASD | 移动 |
